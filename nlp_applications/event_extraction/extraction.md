# 事件抽取/检测

定义：从文本中抽取事件信息。事件信息包括事件的类别以及事件的一些属性。

事件抽取包含触发词识别，事件分类和事件要素识别。
抽取的东西主要包括：  
1）事件类型  
2）触发词  
3）事件描述
4）事件的要素（角色（role）和元素（argument））
事件抽取方法分为pipeline 和joint 两种模式。
